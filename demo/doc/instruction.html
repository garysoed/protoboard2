Protoboard

Protoboard is a library to create board game pieces to be used for prototyping your board games. It also comes with some built in components so you can get started quickly. You can use this demo to play around with some of the pieces.

To begin using this demo, select a component you're interested in from the panel to the left. You will then be able to customize the component in this panel. Use the panel to the right to assemble your components together.
